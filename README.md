# ❌⭕ Tic Tac Toe

A simple, two-player Tic Tac Toe game built for the browser. It’s a great project to understand DOM manipulation and game logic.

## 🎯 Features
- Two-player mode
- Real-time win/draw detection
- Game reset functionality
- Clean and responsive UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🚀 Getting Started
Open `index.html` in your browser to play the game.
